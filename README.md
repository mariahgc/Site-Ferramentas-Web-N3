# Lumina Tech

Projeto desenvolvido para a disciplina de Desenvolvimento Web.

Este site simula uma landing page de uma empresa de energia solar chamada Lumina Tech, com foco em conversão, interatividade e experiência do usuário.

## Funcionalidades

O projeto contém:

- Simulador de economia (ROI)
- Modo sustentável (Dark Mode)
- FAQ interativo
- Formulário de contato com validação
- Cards de produtos
- Design responsivo (mobile e desktop)

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Font Awesome (ícones)
- Google Fonts

## Como executar o projeto

1. Faça o download ou clone o repositório:
2. Abra o arquivo index.html no navegador.
